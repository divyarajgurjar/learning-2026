## Used Tags

1. **`<h1>`**  
   - Used only once to denote my name.

2. **`<h2>`**  
   - Used multiple times to denote headings for different sections.

3. **`<header>`**, **`<main>`**, **`<footer>`**  
   - Used to divide content based on semantic meaning:
     - `header` → Top section  
     - `main` → Primary content  
     - `footer` → Ending section  

4. **`<table>`**, **`<thead>`**, **`<tbody>`**, **`<tr>`**, **`<th>`**, **`<td>`**  
   - Used to create relevant tables.
   - These represent:
     - Table container  
     - Table header  
     - Table body  
     - Table row  
     - Table heading cell  
     - Table data cell  

5. **`<a>`**  
   - Used anchor tags to denote **email** and **telephone number** using special `href` attributes (`mailto:` and `tel:`).
